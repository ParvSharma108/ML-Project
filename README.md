SALARY-PREDICTION-MODEL-ML
💼 Salary Prediction System using Machine Learning

🚀 A Machine Learning project that predicts salaries based on features like experience, education, job role, and skills.

📌 Project Overview This project aims to build and compare multiple machine learning models to accurately predict salaries. It helps in:

📊 Salary estimation for job seekers 🏢 Benchmarking for companies ⚖️ Reducing bias in hiring 🧠 Models Used

The following models were implemented and compared:

Linear Regression Decision Tree Regressor Random Forest Regressor ⭐ (Best Model) Support Vector Regression (SVR) K-Nearest Neighbors (KNN) 📊 Model Performance Model MAE (↓) RMSE (↓) R² Score (↑) Adj R² MAPE (%) Linear Regression 15459 18814 0.5665 0.5530 16.92% Decision Tree 9164 11583 0.8357 0.8306 9.33% Random Forest ⭐ 8978 10929 0.8537 0.8492 9.25% SVR (RBF) 21305 26627 0.1317 0.1047 23.09% KNN 12428 15615 0.7014 0.6921 13.16%

👉 Random Forest Regressor performed best across all metrics
